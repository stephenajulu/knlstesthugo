# 📚 Welcome to the KNLS Website Manager!

This dashboard is your central hub for updating pages, navigation links, and document downloads on the Kenya National Library Service (knls) website.

---

## ✏️ Quick Editing Guide

### 1. How to Edit a Page
1.  Click **Website Pages** in the left sidebar.
2.  Select the page you want to update (e.g. `Strategic Plan`, `Careers`, or `Our Services`).
3.  **Visual Sidebar**: Update titles, descriptions, page heroes, and document lists in the easy-to-use form inputs.
4.  **Content Body**: Type or format your main copy using the visual rich-text editor.
5.  Click the green **Save** button in the top right corner.

### 2. How to Update Menus & Links
1.  Click **Navigation Menus** in the left sidebar.
2.  Open the `navigation.yml` configuration file.
3.  Add, remove, rename, or drag-and-drop navigation items visually.
4.  Click **Save**. The header and footer menus will update automatically.

---

## 🛠️ Adding Content Blocks (Snippets)

While editing page content, you can click the **"+" (Insert Snippet)** button in the formatting toolbar to add pre-designed components:

*   **Button**: Add a link button (select from orange fill, outline, or green colors).
*   **Alert Box**: Insert colored banner notices (useful for library closure notifications or public alerts).
*   **FAQ / Collapsible Accordion**: Add details/summary dropdown questions to easily organize long text.
*   **YouTube Embed**: Embed responsive video players by pasting only the YouTube video ID.

---

## 🚀 Going Live
Every time you click **Save**, CloudCannon:
1.  Commits your updates to the GitHub repository automatically.
2.  Triggers Netlify to compile and deploy your changes.
3.  **The website goes live automatically in about 1-2 minutes!**
