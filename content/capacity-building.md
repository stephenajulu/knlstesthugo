### Strengthening the Library Sector

KNLS coordinates **Capacity Building** programs to support public library development, standard settings, and systems modernization across Kenya.

Following the devolution of public library services, we advise County Governments, install library databases, and run training workshops for librarians. Training modules cover library systems, cataloging metadata, catalog queries, digitization workflows, and visually-impaired support systems.
